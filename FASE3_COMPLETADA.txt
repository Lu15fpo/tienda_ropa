🧮 ALGORITMO DE LUHN (Validación de Tarjetas):

El algoritmo de Luhn es un método de validación de checksums utilizado para
validar números de tarjetas de crédito/débito.

CÓMO FUNCIONA:
1. Recorrer el número de derecha a izquierda
2. Duplicar cada segundo dígito
3. Si el resultado es > 9, restar 9
4. Sumar todos los dígitos
5. Si la suma es divisible por 10, es válido ✓

EJEMPLO: 4532015112830366
6+6+0+6+0+2+2+2+0+2+1+0+1+0+5+8 = 41 → VÁLIDO ✓

📅 VALIDACIÓN DE FECHA DE EXPIRACIÓN:

FORMATO ACEPTADO: MM/YY
VALIDACIONES:
✓ Formato correcto con /
✓ Mes entre 1 y 12
✓ Año no expirado
✓ Conversión automática YY → YYYY (25 → 2025)

EJEMPLOS:
"12/25" → ✓ VÁLIDO (Diciembre 2025)
"01/24" → ✗ INVÁLIDO (Expirado)
"13/25" → ✗ INVÁLIDO (Mes inválido)
"12-25" → ✗ INVÁLIDO (Formato incorrecto)
