[2026-01-05] ERROR #4 - UserController no encontrado (dependencia circular):
  Problema: "UserController" not found cuando ReviewController se inicializa
  Ubicación: GeneralBindings.dependencies() al iniciar la app
  Causa: ReviewController intentaba acceder a UserController.instance en su
         constructor, pero UserController no estaba disponible todavía
  Solución:
    - Cambiado userController en ReviewController de field a getter (lazy loading)
    - Código anterior: final userController = UserController.instance;
    - Código nuevo: UserController get userController => UserController.instance;
    - Cambiado Get.put() a Get.lazyPut() para ReviewController en GeneralBindings
    - Esto evita la inicialización hasta que realmente se use
  Archivos modificados:
    - lib/features/shop/controllers/review_controller.dart
    - lib/bindings/general_bindings.dart
  Estado: ✓ Corregido

[2026-01-05] ERROR #7 - Error persistente de formato de fecha y navegación:

    4. Agregado logging detallado para debug de tipos de datos
