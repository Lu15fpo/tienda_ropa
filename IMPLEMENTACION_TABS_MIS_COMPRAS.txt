```dart
userOrders.sort((a, b) => b.orderDate.compareTo(a.orderDate));
```
• Compara fechas: más reciente (b) vs más antigua (a)
• Resultado: Lista ordenada descendente

FILTRADO:
---------
```dart
// En fetchOrdersByStatus()
if (status == null) {
  return allOrders; // Todos
}

if (status == OrderStatus.processing) {
  return allOrders.where((order) =>
    order.status == OrderStatus.processing ||
    order.status == OrderStatus.pending
  ).toList();
}

return allOrders.where((order) => order.status == status).toList();
```

CONTADORES:
-----------

```dart
// En _getTabCounts()
```

  📝 NOTAS IMPORTANTES

⚠️ AGRUPACIÓN PENDIENTE + PROCESANDO:

   • Se aplica SIEMPRE, en todas las tabs
   • Más reciente primero
   • Basado en orderDate

⚠️ CONTADORES:
   • Se calculan de forma asíncrona
   • FutureBuilder maneja el loading
   • Si falla, muestra 0

⚠️ TABS DESLIZABLES:
   • isScrollable: true
   • tabAlignment: TabAlignment.start
   • Importante para pantallas pequeñas
