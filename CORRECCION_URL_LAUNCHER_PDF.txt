LOGS EN CONSOLA:
```
✅ [OrderController] Abriendo PDF: https://storage.googleapis.com/...
I/UrlLauncher: component name for https://... is null
```

═══════════════════════════════════════════════════════════════
  🔍 ANÁLISIS DEL PROBLEMA
═══════════════════════════════════════════════════════════════

CAUSA:
------
El modo de lanzamiento `LaunchMode.externalApplication` intenta
abrir el PDF directamente con una aplicación externa (visor de PDF).

PROBLEMA:
---------
Android no tiene una aplicación configurada para abrir PDFs
directamente desde URLs de Firebase Storage.

RESULTADO:
----------
`component name is null` = No hay app para manejar esta URL
→ No se abre el PDF

═══════════════════════════════════════════════════════════════
  ❌ CÓDIGO CON PROBLEMA
═══════════════════════════════════════════════════════════════

```dart
final uri = Uri.parse(pdfUrl);
if (await canLaunchUrl(uri)) {
  await launchUrl(
    uri,
    mode: LaunchMode.externalApplication, // ❌ Problema aquí
  );
}
```

INTENTABA:
  • Abrir directamente con app externa (Adobe Reader, etc.)
  • Android busca app para "https://storage.googleapis.com/..."
  • No encuentra → Falla

═══════════════════════════════════════════════════════════════
  ✅ SOLUCIÓN IMPLEMENTADA
═══════════════════════════════════════════════════════════════

CAMBIO:
-------
Usar `LaunchMode.platformDefault` en lugar de `externalApplication`

```dart
final uri = Uri.parse(pdfUrl);
if (await canLaunchUrl(uri)) {
  final launched = await launchUrl(
    uri,
    mode: LaunchMode.platformDefault, // ✅ Solución
  );

  if (!launched) {
    // Mostrar error específico
  }
}
```

CÓMO FUNCIONA:
--------------
• `platformDefault`: Abre en el navegador del sistema
• El navegador descarga el PDF
• El navegador abre el PDF con su visor integrado
• El usuario puede descargar/compartir desde ahí

VENTAJAS:
---------
✅ Funciona en todos los dispositivos Android
✅ No requiere apps específicas instaladas
✅ El navegador maneja la descarga/visualización
✅ Usuario familiar con el flujo

═══════════════════════════════════════════════════════════════
  📊 COMPARACIÓN DE MODOS
═══════════════════════════════════════════════════════════════

LaunchMode.externalApplication:
  • Intenta abrir con app específica
  • ❌ Requiere app instalada
  • ❌ Puede fallar si no hay app
  • Ej: Adobe Reader, Google PDF Viewer

LaunchMode.platformDefault:
  • Abre en navegador del sistema
  • ✅ Siempre funciona
  • ✅ No requiere apps adicionales
  • Navegador maneja el PDF

LaunchMode.inAppWebView:
  • Abre en WebView dentro de la app
  • ✅ Funciona
  • ❌ UI no nativa
  • Para casos específicos

LaunchMode.externalNonBrowserApplication:
  • Similar a externalApplication
  • ❌ Mismo problema
