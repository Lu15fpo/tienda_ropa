PASO 9.1.1: GUARDAR PDF EN FIREBASE STORAGE (NUEVO)
-------------------------------------------------------------

✅ Se agregó funcionalidad para guardar el PDF generado en Storage:

1. Ruta del archivo: Facturas/{numeroFactura}_{claveAcceso}.pdf
   Ejemplo: Facturas/001-001-000000025_1201202601...pdf

2. Metadata incluida:
   - contentType: application/pdf
   - orderId: ID del pedido
   - claveAcceso: Clave de acceso de 49 dígitos
   - numeroFactura: Número formateado (001-001-000000025)

3. Archivo público: ✅
   - El archivo se hace público automáticamente
   - URL de acceso directo generada

4. URL guardada en Firestore:
   - Campo: pdfUrl
   - Colección: Facturas
   - Documento: {claveAcceso}

CÓDIGO AGREGADO:
```javascript
// Guardar PDF en Firebase Storage
const bucket = admin.storage().bucket();
const numeroFactura = `${establecimiento}-${puntoEmision}-${secuencial}`;
const pdfFileName = `Facturas/${numeroFactura}_${claveAcceso}.pdf`;
const pdfFile = bucket.file(pdfFileName);

await pdfFile.save(pdfBuffer, {
  metadata: {
    contentType: 'application/pdf',
    metadata: {
      orderId: orderId,
      claveAcceso: claveAcceso,
      numeroFactura: numeroFactura,
    },
  },
});

// Hacer el archivo público
await pdfFile.makePublic();

// Obtener URL pública
pdfUrl = `https://storage.googleapis.com/${bucket.name}/${pdfFileName}`;
```

1. SERVICIO DE FACTURACIÓN (facturacion_service.dart)
-------------------------------------------------------------

✅ Método: obtenerUrlPdfFactura(String facturaId)
   • Obtiene la URL del PDF desde Firestore
   • Parámetro: ID de la factura (clave de acceso)
   • Retorna: URL del PDF o null

✅ Método: obtenerUrlPdfPorPedido(String orderId)
   • Obtiene la URL del PDF a partir del ID del pedido
   • Busca en la colección "Facturas" por orderId
   • Retorna: URL del PDF o null

CÓDIGO:
```dart
Future<String?> obtenerUrlPdfPorPedido(String orderId) async {
  final facturas = await FirebaseFirestore.instance
      .collection('Facturas')
      .where('orderId', isEqualTo: orderId)
      .limit(1)
      .get();

  if (facturas.docs.isEmpty) return null;

  return facturas.docs.first.data()['pdfUrl'] as String?;
}
```

2. CONTROLADOR DE PEDIDOS (order_controller.dart)
-------------------------------------------------------------

✅ Método: verPdfFactura(String orderId)
   • Muestra loader "Obteniendo factura..."
   • Obtiene URL del PDF usando el servicio
   • Abre el PDF en el visor del sistema
   • Maneja errores y mensajes al usuario

FLUJO:
1. Usuario presiona botón "Ver Factura PDF"
2. Se muestra loader
3. Se consulta Firestore por la URL del PDF
4. Si existe, se abre con url_launcher
5. Si no existe, se muestra mensaje informativo
6. Se cierra el loader

CÓDIGO:
```dart
Future<void> verPdfFactura(String orderId) async {
  // Obtener URL
  final pdfUrl = await facturacionService.obtenerUrlPdfPorPedido(orderId);

  if (pdfUrl == null) {
    TLoaders.warningSnackBar(
      title: 'Factura no disponible',
      message: 'La factura aún no ha sido generada.',
    );
    return;
  }

  // Abrir PDF
  final uri = Uri.parse(pdfUrl);
  await launchUrl(uri, mode: LaunchMode.externalApplication);
}
```
