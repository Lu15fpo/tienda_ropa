📱 FLUJOS DE USUARIO IMPLEMENTADOS:

1. AGREGAR MÉTODO DE PAGO
   Perfil → Mis Métodos → [+] → Formulario → Guardar → Lista actualizada

2. USAR EN CHECKOUT
   Carrito → Pagar → Método predeterminado se carga → Cambiar (opcional) → Pagar

3. ESTABLECER PREDETERMINADO
   Mis Métodos → Tap en tarjeta → Badge "Predeterminado" se actualiza

4. ELIMINAR MÉTODO
   Mis Métodos → [🗑️] → Confirmar → Lista se actualiza

5. AGREGAR DESDE CHECKOUT
   Checkout → Cambiar → Agregar nuevo → Formulario → Guardar → Regresar
